  1.  new Network(_dimension: Dimension, _dataStructure: AdjacencyStructure)

    A Network Projection.

    A Network Projection. Inherits from Projection.

    _dimension

    the dimension of the Network. Unit is in points.

    _dataStructure

    The DataStructure of choice. AdjacencyMatrix and AdjacencyList available.

    val context = new Context()

context.addAgent(a1)
